# 智能问答Agent系统 ———— Agent_Q
### 本人是一名初二在校生，写代码实属不便，有大佬可以更正一下我的细节吗？谢谢
基于DeepSeek API的现代化AI问答系统，具备自主学习能力和专业的数据分析功能。

## 🌟 核心功能

### 智能聊天问答
- 基于DeepSeek API的智能对话
- 实时流式响应，打字机效果
- 上下文记忆和多轮对话
- 用户反馈和自主学习

### 知识库管理
- 分类管理知识条目
- 全文搜索和筛选
- 数据可视化和统计
- 批量导入导出

### 数据分析中心
- 实时性能监控
- 用户行为分析
- 问答质量评估
- 学习进度追踪

### 安装依赖
```bash
npm install
```
